# Petrobras AI Challenge — Site Institucional

## 1. Descrição do Projeto
Site institucional sobre o desafio tecnológico da Petrobras focado em acesso seguro a sistemas corporativos de IA por meio de óculos de realidade aumentada (smart glasses). O site apresenta o contexto do desafio, o problema central, os serviços/soluções propostas, galeria visual e formulário de contato.

## 2. Estrutura de Páginas
- `/` — Página única com âncoras:
  - `#home` — Hero / Início
  - `#about` — Sobre o Desafio
  - `#services` — Serviços / Soluções
  - `#gallery` — Galeria
  - `#contact` — Contato

## 3. Funcionalidades Principais
- [x] Navbar com logo e navegação por âncora
- [x] Seção Hero com destaque do desafio
- [x] Seção About com contexto da empresa e problema central
- [x] Seção Services com soluções propostas
- [x] Seção Gallery com imagens do projeto
- [x] Seção Contact com formulário funcional
- [x] Footer com informações e links

## 4. Modelo de Dados
Sem banco de dados necessário. Formulário de contato usa integração via Readdy Forms.

## 5. Integrações / Backend
- Supabase: Não necessário
- Shopify: Não necessário
- Stripe: Não necessário
- Readdy Forms: Formulário de contato (https://readdy.ai/api/form/d7h2espaljjjkv6jg2u0)

### Fase 1: Construção do Site Completo
